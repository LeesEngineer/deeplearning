# SoftMax regression

</br>

<p>如果有多种可能的分类目标，有一种更普遍的逻辑回归方法叫做 softmax 回归</p>

<p>如果要识别四种类别的物品</p>

`C = #class = 4`

<p>在这种情况下，需要构建一个输出层是 C 的神经网络，n[L] = C 。想得到的输出层的单元告诉我们每个类别的概率 P(object | x)，输出的概率加和应该是 1，标准化的做法是使用 softmax 层</p>

```
z[L] = W[L] a[L - 1] + b[L] // (4, 1) vector
a[L] = g(z[l])
"""
Activation function:
    t = e^{(z^{[L]})}
    a^{[L]} = \frac{t_i}{\sum^4_{i = 1}{t_i}}
"""
```
